System Requirements

To ensure that the ROCK IMAGER FT software runs smoothly, your computer must meet the following minimum requirements. To be able to use the software on your personal device, a computer server must be set up. This section specifies what the computer requirements are for both the server’s and client’s computers.

Note: If you need further assistance related to the system requirements, please contact support@formulatrix.com.

Client's Requirements

ROCK IMAGER FT 4.0
Recommended
Operating System Linux Ubuntu 18.04 LTS 64-bit
Browser Support Google Chrome (recommended), Mozilla Firefox.
Processor Speed Quad Core 3.6 GHz
RAM 16 GB
Screen Resolution 1920 x 1080 pixels

Server's Requirements

ROCK IMAGER FT 4.0
Recommended
Operating System Linux Ubuntu 18.04 LTS 64-bit
Processor Speed Quad Core 3.6 GHz
RAM 16 GB
.NET Framework .NET Framework 4.7 and .NET Core 2.2
Database PostgreSQL 12.0
