Tools Menu

Many tools in the ROCK IMAGER FT highly affect the system. The Tools menu contains run sequences in the hardware that you can perform in order to make sure whether they work well or not to maintain the maximum performance of the ROCK IMAGER FT system.

Important: Only advanced users or FORMULATRIX support are allowed to modify any settings in this menu!
