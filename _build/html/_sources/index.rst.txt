Welcome to ROCK IMAGER FT 4.1's documentation!
==============================================
Welcome to the help file for the ROCK IMAGER FT (Flexible Temperature) 260, an automated system for imaging crystallization plates on a user-defined schedule. This help center contains detailed information on how to use ROCK IMAGER FT software and hardware, starting from loading and unloading plates to performing more advanced plate management tasks, including fine-tuning drop location on a per-plate basis and applying image masks.
